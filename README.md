# Использование
## client
```
Использование: client <имя_файла>
```

### Отправка данных при рабочем сервере
**Команда**
```
cargo run -p client .\test.txt
```

**Результат**
```
Ответ от сервера:
Имя файла: uploaded_1744357188.txt
Строк: 3
Слов: 9
Символов: 59
```

## server
**Запуск сервера**
```
cargo run -p server
```

**Консоль:**
```
Сервер запущен на 127.0.0.1:8080
```


Для файла:
```txt
Тестовый файл
состоящий из 3х строк
для отправки серверу.
```
